# Task 1

Created a folder called GitHub_task and added a Hello World C++ program using Linux terminal and pushed it to GitHub.

## Linux Commands Used:
- mkdir GitHub_task → create directory  
- cd GitHub_task → enter directory  
- nano hello.cpp → create/edit file  
- g++ hello.cpp -o hello → compile program  
- ./hello → run program  

## Git Commands Used:
- git init → initialize repo  
- git remote add origin <repo-url> → connect to GitHub  
- git add . → stage files  
- git commit -m "message" → save changes  
- git branch -M main → set branch  
- git push -u origin main → upload to GitHub
